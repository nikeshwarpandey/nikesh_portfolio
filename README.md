# Dev Portfolio

A modern personal portfolio website built with React and Vite, designed to showcase software development experience, featured projects, testimonials, and contact information in a polished single-page experience.

## Highlights

- Responsive hero and about sections
- Project showcase with interactive cards
- Experience timeline and testimonials carousel
- Contact section with a clean, professional layout

## Tech Stack

- React
- Vite
- Lucide React
- CSS-based styling and animations
